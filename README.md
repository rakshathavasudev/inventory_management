# AI Operations App

A full-stack web application for managing product orders, viewing mockups, and generating shipping labels.

This project demonstrates how an order moves through a simple fulfillment workflow â€” from creation to approval to shipping label generation.

---

## Features

- View customer orders  
- View product mockups  
- Approve orders for fulfillment  
- Enter shipping details  
- Generate printable shipping labels (PDF)  
- Barcode automatically added to each label  

---

## Tech Stack

### Frontend
- React  
- TypeScript  
- Vite  
- React Router  

### Backend
- Go  
- Gin Framework  
- gofpdf (PDF generation)  
- Code128 barcode generator
